void ControlCenter::menu ( void  )

change /etc/syscrl.conf, in order to avoid the TIME_WAIT state by ports

these four scripts should be changed

net.ipv4.tcp_syncookies = 1

net.ipv4.tcp_tw_reuse = 1

net.ipv4.tcp_tw_recycle = 1

net.ipv4.tcp_fin_timeout = 30

other things can not be changed the menu of ControlCenter
